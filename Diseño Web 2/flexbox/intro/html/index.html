Formato visual

Es la forma mediante la cual los elementos se acomodan dentro de la página.

Existen dos formas principales en las que los elementos se pueden acomodar:

Elementos en línea
Son aquellos que se acomodan uno al lado del otro. Por ejemplo los elementos de enlace, input, label, span, etc.
Elementos en bloque
Son aquellos elementos que se acomodan por defecto uno abajo del otro y ocupan el 100% del ancho de la página.

Adicionalmente, existen otras posibilidades de modificar el formato visual con los valores inline-block, list, table, etc.

Flexbox

Es el formato visual que se ha incluido más recientemente y permite crear páginas web más fluidas y además provee múltiples facilidades para acomodar los elementos dentro de una página.
